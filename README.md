#  Project Overview

An interactive **Fashion Retail Analytics Dashboard** built with **Power BI** to analyze sales performance, profitability, advertising efficiency, customer behavior, and seasonal business trends for a fashion boutique business.

This project transforms raw retail data into actionable business insights through dynamic visualizations, KPI monitoring, and modern dashboard design.

The dashboard provides comprehensive analysis across:

- Revenue & Profit Analysis
- Advertising Spend Monitoring
- Seasonal Sales Performance
- Marketing Channel Analysis
- Customer Type Insights
- Service Type Distribution
- Monthly Sales Trends
- Interactive Business Reporting

---

#  Project Objectives

- Analyze fashion retail sales performance
- Monitor revenue, profit, and profit margin KPIs
- Evaluate advertising spend effectiveness
- Identify top-performing marketing channels
- Track seasonal business trends
- Improve business decision-making using data visualization
- Build professional and interactive Power BI dashboards
